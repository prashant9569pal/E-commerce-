## Run Locally

Clone the project

```bash
  git clone https://github.com/prashant9569pal/E-commerce-?tab=readme-ov-file
```

Go to the project directory

```bash
  cd React_E-Commerce
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
